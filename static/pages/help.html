Quickstart
Here's a quick runthrough of ZephyrPlus.

ZephyrPlus and Zephyr:
ZephyrPlus is a web client for Zephyr, a messaging system. Zephyr has a unique culture, which you can read about here __link___. For one, please type in complete sentences.

Classes and Instances:
In Zephyr, Classes are like rooms and instances are like subjects. You subscribe to a class in order to receive and send messages to that class. classes have instances, which describe the subject of discussion. Instances change regularly as users make new instances.

Link Bar:
The Link Bar displays your username and has links to pages on the site.

Classes Header:
You can subscribe to a class by clicking "add a class," to the right. Clicking "classes" will change your view to display messages from all classes.

Classes List:
The Classes List is the list of classes that you are subscribed to. You can view all messages from a class by clicking on a class name. If there are unread messages in a class, the name will be bolded and have a number in parenthesis. You are automatically subscribed to the "lobby" class and your own class. You can click the triangle to toggle the Instances List (see below).

Instances List:
The Instances List is the list of current instances for the parent class. You can view all messages from an instances by clicking on the instance name. Again, the list of instances will change regularly as discussions change.

Chat Header:
The Chat Header displays what messages are currently displayed below. Here, messages from "all classes" are displayed. If you clicked on "lobby" class, the Chat Header would change to "all classes > lobby" and only messages from the lobby class would be displayed. Clicking on "all classes" will bring you back to displaying all messages."

Message Header
Each message has a Message Header. The format is "[Class] / [Instance] / [Sender]." The message text is immediately below.

Message Timestamp:
When you hover on a message, the time the message was sent is shown on the right.

Chat Bar:
The Chat Bar displays what class and instance you are sending your message to. You can set the class and instance by using the dropdown menus. Clicking a message will set your Chat Bar to the same class and instance as the message, allowing easy replies.

Message Box:
You can enter your message text into the Message Box. You can send a message by pressing send in the Chat Bar. You can also use "dot sending," which sends the message when you type a dot by itself on a line and press enter.

FAQ

How do I send a personal?
We currently do not support personal messages.

Can I see who is in a class?
No, Zephyr does not support viewing the members of a class. It's one of the unique aspects of Zephyr!

I have a great bug and/or feature request! Now what?
That's great, we love suggestions! You can contact us at zephyrplus@mit.edu or on the zephyrplus class.

For any other questions, please contact us at zephyrplus@mit.edu or on the zephyrplus class.
