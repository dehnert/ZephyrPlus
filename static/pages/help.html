Welcome to ZephyrPlus! ZephyrPlus is an interface to MIT's command line chat system, Zephyr. ZephyrPlus lets you send messages to or subscribe to classes-- A class is a topic or thread of related topics that anyone can write to. Sometimes you may want to write to a specific part of a class or a specific subtopic that falls in a class. You may do this by writing to an instance of a class. More help content...
